HOW TO ADD REAL TEMPLE PHOTOS
==============================

Save photos in this folder using these exact filenames. Once a file exists,
it will automatically appear on the website (replacing the illustration).

  birla-mandir.jpg     - Birla Mandir, Khairtabad
  chilkur-balaji.jpg   - Chilkur Balaji Temple
  jagannath.jpg        - Jagannath Temple, Banjara Hills
  karmanghat.jpg       - Karmanghat Hanuman Temple
  peddamma.jpg         - Peddamma Temple, Jubilee Hills
  sanghi.jpg           - Sanghi Temple
  ujjaini.jpg          - Ujjaini Mahankali Temple
  keesaragutta.jpg     - Keesaragutta Temple

TIPS
----
- Recommended size: around 800 x 500 pixels (landscape)
- JPG works best, but .png and .webp will also work if you change the
  extension in the HTML's img src tags
- Photos should be ones you have permission to use (your own pictures,
  Wikimedia Commons, or licensed stock photos)

WHERE TO FIND FREE-TO-USE TEMPLE PHOTOS
---------------------------------------
- Wikimedia Commons (commons.wikimedia.org) - check the license on each photo
- Telangana Tourism website
- Your own photos from a temple visit
